# stpyapp-template
[streamlit](https://streamlit.io/)のアプリ開発するためのテンプレートを作ってみる


## Usage
### Setup
```sh
poetry install
poetry shell
```

### Start as local service
```sh
# on poetry shell
# streamlit hello
streamlit run src/main.py
# Local URL: http://localhost:8501
```
